## **Описание проекта** 📜

Данная программа была написана в качестве расчётно-графического задания для 1 курса по дисциплине "Основы программирования". В ней делался упор на закрепление принципов объектно-ориентированного программирования.

---

🧩 ~Блок-схема программы находится в файле [block-scheme.png](https://github.com/m0zerrr/exchange-rate/blob/main/block-scheme.png) ~

🗄️ ~Тестовый файл с датасетом находится в файле [CZK_data.csv](https://github.com/m0zerrr/exchange-rate/blob/main/CZK_data.csv) ~

---

При запуске "приложения" мы можем получить курс валюты
на сегодняшний день или на любую желаемую дату.

Немного ниже предлагается загрузить датасет с локального устройства (не менее 5 лет) или выгрузить официальные данные с сайта Центрального банка РФ, данные при этом будут загружены с 1 января 2021 года.

`‼️ВАЖНО: при выгрузке данных с сайта ЦБ в папке с кодом будет создан файл формата CSV!`

После вывода уведомления об успешной загрузке данных можно перейти к основной части работы - <u>анализу валюты</u>. В окошке сразу выводится минимум и максимум за исследуемый период, а также средний курс за каждый год.

`‼️Результаты также сохраняются в CSV-файл!`

Также программа позволяет построить графики: за весь наблюдаемый срок или за выбранный, диаграмму средних курсов по годам.

`‼️Построенные графики сохраняются на локальном устройстве!`

После выполнения программы есть возможность сохранить все полученные файлы в ZIP-папке в любое место на компьютере, или в облако Яндекс.

`В коде токен для облака Яндекс отсутствует. Его необходимо получить по ссылке:` https://oauth.yandex.ru/client/new
